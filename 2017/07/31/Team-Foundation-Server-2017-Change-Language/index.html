Team Foundation Server 2017 Change Language

Posted on 2017-07-31   |   In CI&CD   |     |  

分享如何切換 TFS Web Access的語系顯示

當我們一開始登入 TFS 服務後，會發現 TFS 的服務會呈現我們預設的網頁瀏覽器語(browser preferred language)系或是當前使用的系統偏好語系。

如果要更改呈現語系，有兩種做法:

  1. 修改 TFS 使用者檔案
  2. 修改瀏覽器預設語系

修改 TFS 使用者檔案

  1. 開啟 Team Web Access Portal

  2. 點選右上角的使用者名稱

  3. 右鍵選擇 我的設定檔

  4. 點選 locale 分類項(tab)，選擇語系與相關設定

  5. 儲存後，頁面會自動更新。即可看到更新後的指定語系呈現

修改瀏覽器預設語系

當然我們也可以透過修改瀏覽器預設語系的方式達到相同效果，這邊可以參考該文章的整理

How to Change Your Browser’s Language

如果是 Chrome 的愛好者，可以參考Quick Language Switcher 這個套件幫你做語系切換的功能。
